# ✦ TaskFlow

> Get things done, beautifully.

A dark editorial task manager built with **React + Vite** — gold accents, due dates, filters, and smooth animations.

## 📸 Preview

![TaskFlow Preview](./preview.png)

## ✨ Features

- ✅ Add, complete, and delete tasks
- 📅 Due dates with Overdue / Today / Tomorrow labels
- 🔍 Filter by All / Active / Completed
- 📊 Live progress bar
- 🌑 Dark editorial UI with gold accents
- ✨ Smooth entrance animations
- 📱 Fully responsive

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/pujanrasaili/taskflow.git

# Go into the folder
cd taskflow

# Install dependencies
npm install

# Start dev server
npm run dev
```

## 🛠 Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- Pure CSS (no UI libraries)
- CSS custom properties for theming

## 📁 Project Structure

src/
├── App.jsx       # Main component & logic
├── App.css       # Component styles
├── index.css     # Global styles & design tokens
└── main.jsx      # Entry point

## 🌙 Dark Mode

TaskFlow automatically switches to dark mode based on your system preference using `prefers-color-scheme`.
